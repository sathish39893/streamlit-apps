# streamlit-apps
streamlit apps built using generative ai llms
## Getting Started
To run the streamlit apps, follow these steps:
1. Clone the repository:
```bash
git clone https://github.com/sathish39893/streamlit-apps.git
``` 
2. Navigate to the project directory:
```bash
cd streamlit-apps
```
3. Install the required dependencies:
```bash
pip install -r requirements.txt
```
4. Run the streamlit app:
```bash
streamlit run main.py
```
5. Open your web browser and go to `http://localhost:8501` to view the app.
